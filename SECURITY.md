# Security Policy

## Supported Versions

This is a static personal contact page. There is no versioned release cycle.

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it
responsibly by emailing:

**contact-us@alexander.xin**

Please include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact

I will acknowledge the report within **2 business days** and work to address
confirmed issues promptly. Please do not open a public GitHub issue for
security-related matters.

## Security Measures in This Project

- **Cloudflare Turnstile** is used as a bot verification gate to prevent
  automated email harvesting.
- No user data is collected or stored server-side.
- All external resources are loaded over HTTPS.
